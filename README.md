# Metadata Labels

Metadata Labels is an Obsidian plugin scaffold for frontmatter-driven note labels.

This scaffold intentionally contains no feature implementation yet. Planned functionality includes watching frontmatter metadata, supporting configurable rules, and applying note icons and colours in the File Explorer based on fields such as Editing Status, Editing Stage, or any configured metadata field.

## Development

Install dependencies:

```bash
npm install
```

Start the development build:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Lint the project:

```bash
npm run lint
```
